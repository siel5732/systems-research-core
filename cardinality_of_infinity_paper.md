### 2.1 The Classical Shift Operator
If a single new guest arrives, the hotel manager simply moves the guest in room $n$ to room $n+1$:

$$|n\rangle \to |n+1\rangle$$

This frees up Room 1. If an infinite number of new guests arrive, the manager moves the guest in room $n$ to room $2n$:

$$|n\rangle \to |2n\rangle$$

This frees up all odd-numbered rooms ($1, 3, 5, \dots$), which are countably infinite ($\aleph_0$), allowing the infinite new arrivals to be accommodated instantly without displacing any existing guests.
